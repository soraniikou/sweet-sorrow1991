🐦️sweet sorrow- 苦しいことあっても生きていれば嬉しい瞬間もある




Listenを押すと、小川と、小鳥の声が流れます。

最初は、地震で通信が途絶えた人のためのアプリを作ろうとしていました。

一度聞けばオフラインでも動作。

しかし災害のあとに他人の声や小鳥の声を聴く余裕はないから、一度は公開やめようとしました。

それでも公開することにしたのは、
災害ではなく、ふつうの夜の、ふつうの孤独のためなら、
意味があるかもしれないと思ったからです。

苦手なことがあっても生きていける。というメッセージを入れて作りました。私には苦手がたくさんあるからです。


1分で終わります。※鳥のさえずりは効果音ラボさん（https://soundeffect-lab.info/）の素材を使わせていただきました。

https://sweet-sorrow1991.vercel.app
